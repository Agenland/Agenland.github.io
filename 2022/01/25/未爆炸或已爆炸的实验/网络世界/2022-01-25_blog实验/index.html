# Blog 实验小指南（hexo + github）

小于 5 度

# 软件安装与基本设置，自定义域名。

这一方面已经有很多作者说的非常详细了，我就推荐两个教程。

  • 图文教程 GitHub+Hexo 搭建个人网站详细教程 - 吴润的文章 - 知乎

这篇文章把 hexo 搭建的主线都讲出来了，另外还讲了如何使用自定义域名。

后面有讲关于主题和 md 的，如果你不用 next 主题就可以选择性观看。

注意：由于 GitHub 更新，_config.yml 里的 branch 后面不是填 master，填 main。

​

  • 视频教程 保姆级 Hexo+github 搭建个人博客 @视频搬运崽啊

这个视频按最近 github 的更新改动过，比较可靠。

​

（补充）阿里云上注册域名有三步：

  1. 购买一个域名。
  2. （模板界面操作）将自己的模板实名认证。（模板的实名认证提交以后大概 2~3 天后会完成，会有短信通知）
  3. （域名列表界面操作）把这个模板提交给你刚注册的域名再认证一次。（不到一天会完成，会有短信通知）

当域名状态显示正常的时候就可以使用了。

​

# 寻找自己喜欢的主题

​ 基本上所有的 hexo 主题都会发布到 github 上，直接在 github 上搜索 hexo theme 就可以找到很多主题了，他们一般会附上相关的网页效果链接（叫做 Demo/pre/preview 之类）。

​

​ 当然如果你没有时间一个个找，你也可以看看我推荐的这些有没有喜欢的（点击可进入网页效果展示）:

​

shoka (强，烈，推，荐。好看功能全)

bufferfly fluid (两个简洁风格)

常・也 (非，常，好，看)

PPOffice (这是一篇国外的主题，也很好看)

云游君的小站 （好看，作者内心 os 特别多）

nexmoe (很有个性，归档页面的分类是少有的好看，另外，作者在里面写了很多很多很多文章)

3-hexo （三段式的理念是我想要的，奈何就是界面有一点点的，嗯）

​

# 学习主题的使用

​

​ 可以在 https://hexo.io/themes/ 上寻找，或者去 github 搜索相关主题。通常在 github 里的 Readme 文档里会给出官方文档的位置，依照官方文档学习就行了。
你可能会需要先看一下 hexo 的官方文档，有些作者会默认你已经看过了，然后不会再介绍相关内容。

​

如果你是第 1 次使用，建议从 Butterfly 开始，因为它简单，而且官方文档全，用的人也多（代表着各种 B 站知乎等等相关的教程也会很多）。
我使用的是 shoka，因为它的分类是最舒服的，而且带有音乐等好玩的功能。

​

# 小工具推荐

​

​ hexo 有相当多的插件 / 轮子，推荐 hexo-enhancer。可以自然的生成标签，分类，日期名称等信息。详细介绍可以在 Hexo 功能增强插件查看。
